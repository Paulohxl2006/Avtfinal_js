Atvidade final de html - Curso de Robotica

A atividade de Desenvolvimento de Sistemas Web em HTML/HTML5 foi realizada com o tema "Jogos Antigos". A página foi estruturada usando tabelas para organizar textos, imagens com legendas, e um formulário interativo. A formatação de texto foi aplicada para destacar informações importantes. A página possui três links principais (HOME, SOBRE, CONTATO) e está disponível como uma página estática através do GitHub Pages. Todo o projeto foi organizado nesse repositório no GitHub.

https://paulohxl2006.github.io/Atvfinal_html/

Produtos ivone - Curso de Robotica

Site de produtos para cabelo, especificamente voltado para diferentes tipos de cabelo (Cacheado, Liso, Crespo). O layout inclui uma imagem de fundo com flores e uma barra de navegação no topo, com links para "Home", "Contato", e "Sobre nós". A pagina tem produtos como shampoo, condicionador e máscara, com seus respectivos preços. A página utiliza uma paleta de cores suaves, com predominância de rosa claro e branco, destacando os produtos e suas descrições.

https://paulohxl2006.github.io/Atv_05-css/

Poke API - Curso de Robotica

Esta página é uma interface para buscar informações sobre Pokémon usando uma API. O layout apresenta um campo de entrada onde o usuário pode digitar o nome de um Pokémon, seguido de um botão "enviar" para realizar a busca. Após a pesquisa, as informações do Pokémon, como tipo, habilidades, ataque especial, defesa especial, entre outras estatísticas, são exibidas na tela junto com uma imagem do Pokémon correspondente. O cabeçalho da página contém o logo da franquia Pokémon, dando um toque temático à interface.

https://paulohxl2006.github.io/atv01_js/
